# Star Wars Timeline

Este trabalho apresenta uma linha do tempo em HTML inspirada no universo de *Star Wars*.  
O projeto reúne imagens dos personagens e organiza o conteúdo em uma página simples para visualização no navegador.

## Conteúdo do projeto

- `HTML STAR WARS/star_wars_timeline.html`: página principal do trabalho
- `HTML STAR WARS/download_images.ps1`: script para baixar imagens usadas no projeto
- `pesquisa_web.md`: material de apoio com pesquisa
- `pesquisa_web.pdf`: versão em PDF da pesquisa

## Como abrir

1. Abra a pasta `HTML STAR WARS`.
2. Dê um duplo clique em `star_wars_timeline.html`.
3. A página será aberta no navegador padrão.

## Objetivo

- Apresentar personagens marcantes de `Star Wars`
- Organizar o conteúdo em uma linha do tempo visual
- Usar imagens para facilitar a identificação dos personagens

## Observações

- O projeto foi desenvolvido em HTML.
- Se as imagens não aparecerem, verifique se elas estão na mesma pasta do arquivo HTML.
